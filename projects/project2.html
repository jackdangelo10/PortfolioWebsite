Description

In collaboration with my professor at Georgetown University, who is supported by a NASA grant, I am contributing to the ongoing development of a WebGL-driven application to visualize terrain data from NASA's MOLA (Mars Orbiter Laser Altimeter) and LOLA (Lunar Orbiter Laser Altimeter) missions. My focus is on the backend, handling data partitioning, fetching, and implementing persistent storage solutions. This project aims to create an interactive 2D terrain visualization tool as a web application, allowing NASA planetary geologists to explore detailed topographical data of Mars and the Moon. We are consulting with a NASA planetary geologist for advice on use cases, UI, and functionality, enhancing the accessibility and usability of planetary data for scientific analysis and educational purposes. The first version of the application is scheduled for release in January.
